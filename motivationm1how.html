2. ຄວາມຮູ້ສຳລັບຮອບສຳພາດ (Interview)

ຂັ້ນຕອນການຄັດເລືອກ

ສຳລັບຮອບຄັດເລືອກ ແມ່ນຈະມີທັງໝົດ 2 ຮອບຄື: ຮອບສອບເສັງພາກຂຽນ ແລະ ຮອບສຳພາດ. ທັງສອງຮອບ ແມ່ນຈະຈັດຢູ່ສູນກາງຊາວໜຸ່ມປະຊາຊົນປະຕິວັດລາວ (ນະຄອນຫຼວງວຽງຈັນ). ສຳລັບລາຍລະອຽດການສະໝັກ ແລະ ສອບເສັງແມ່ນມີຢູ່ 3 ຂັ້ນຕອນດັ່ງນີ້:

1. ຮັບເອົາແບບຟອມສະໝັກ, ຕື່ມຂໍ້ມູນຕົນເອງໃສ່ ແລະ ຍື່ນກັບຄືນໃຫ້ສູນກາງຊາວໜຸ່ມປະຊາຊົນປະຕິວັດລາວ (ນະຄອນຫຼວງວຽງຈັນ).

-> ເອກະສານຕິດຄັດຕ່າງໆ ແມ່ນປະກອບມີ: 1) ສຳເນົາໃບຢັ້ງຢືນເຂົ້າຊາວໜຸ່ມ (ສາມາດຍື່ນສຳເນົາໃບຍົກຍ້າຍຊາວໜຸ່ມໄດ້ໃນກໍລະນີບໍ່ມີໃບຢັ້ງຢືນເຂົ້າຊາວໜຸ່ມ ຫຼື ໃບຢັ້ງຢືນຈາກບ້ານໃນກໍລະນີບໍ່ໄດ້ເປັນສະມາຊິກຊາວໜຸ່ມ) ແລະ 2) ສຳເນົາໃບຍ້ອງຍໍ, ສຳເນົາໃບຢັ້ງຢືນການເຂົ້າຮ່ວມໂຄງການຕ່າງໆ ຫຼື ສຳເນົາໃບປະກາສະນີຍະບັດຈົບຫຼັກສູດຕ່າງໆ.

-> ໄລຍະເວລາເປີດສະໝັກຍື່ນຟອມ ແມ່ນປະມານ 1 ອາທິດ ໃນຊ່ວງໃດໜຶ່ງຂອງເດືອນພຶດສະພາ.

-> ລໍຖ້າໄລຍະເວລາປະມານ 1 ອາທິດຫຼັງຈາກປິດຮັບຟອມສະໝັກ. ທາງ facebook fanpage ຂອງສູນກາງຊາວໜຸ່ມປະຊາຊົນປະຕິວັດລາວ, ຂອງ Lao Youth Radio FM 90.0 Mhz, ຂອງອະດີດຜູ້ເຂົ້າຮ່ວມໂຄງການ ແລະ ອື່ນໆ ຈະແຈ້ງການກ່ຽວກັບລາຍຊື່ຜູ້ສອບເສັງ.

2. ເຂົ້າຫ້ອງສອບເສັງຢູ່ທີ່ສູນກາງຊາວໜຸ່ມປະຊາຊົນປະຕິວັດລາວ (ນະຄອນຫຼວງວຽງຈັນ). ສຳລັບການກະກຽມ ແມ່ນສາມາດເຂົ້າເບິ່ງໄດ້ໃນພາກທີ 1 (ຄວາມຮູ້ສຳລັບຮອບເສັງຂຽນ).

-> ຫົວບົດສອບເສັງ ແມ່ນເປັນພາສາອັງກິດ.

-> ຄວາມຮູ້ທີ່ຕ້ອງມີ: 1) ຄວາມຮູ້ກ່ຽວກັບປະເທດລາວ, 2) ຄວາມຮູ້ກ່ຽວກັບຄະນະຊາວໜຸ່ມປະຊາຊົນປະຕິວັດລາວ, 3) ຄວາມຮູ້ກ່ຽວກັບ ASEAN ແລະ ຍີ່ປຸ່ນ, 4) ຄວາມຮູ້ກ່ຽວກັບໂຄງການຊາວໜຸ່ມເດີນທະເລ ແລະ 5) ທັກສະພາສາອັງກິດ.

-> ຄຳແນະນຳແມ່ນ ອ່ານໃຫ້ຫຼາຍ, ຟັງໃຫ້ຫຼາຍ ແລະ ອ່ານໃຫ້ຖືກຈຸດ, ຟັງໃຫ້ຖືກຈຸດ. ເຮັດຫຍັງກໍໃຫ້ເຮັດຢ່າງຄົນມີເຕັກນິກ.

-> ລໍຖ້າຜົນສອບເສັງປະມານ 1 ອາທິດ. ທາງ facebook fanpage ຂອງສູນກາງຊາວໜຸ່ມປະຊາຊົນປະຕິວັດລາວ, ຂອງ Lao Youth Radio FM 90.0 Mhz, ຂອງອະດີດຜູ້ເຂົ້າຮ່ວມໂຄງການ ແລະ ອື່ນໆ ຈະແຈ້ງການກ່ຽວກັບລາຍຊື່ຜູ້ມີສິດສຳພາດ.

3. ເຂົ້າຫ້ອງສຳພາດຢູ່ທີ່ສູນກາງຊາວໜຸ່ມປະຊາຊົນປະຕິວັດລາວ (ນະຄອນຫຼວງວຽງຈັນ). ສາມາດຊອກຂໍ້ມູນອ້າງອີງໄດ້ໃນພາກທີ 2 (ຄວາມຮູ້ສຳລັບຮອບສຳພາດ).

-> ລັກສະນະຄຳຖາມຈາກຄະນະກຳມະການ ສາມາດແບ່ງເປັນ 4 ພາກໄດ້ດັ່ງນີ້: 1) ໃຫ້ແນະນຳກ່ຽວກັບຕົນເອງ, 2) ໃຫ້ຈົກສະຫຼາກເພື່ອເລືອກຫົວຂໍ້ໜຶ່ງ (ຈາກທັງໝົດມີປະມານ 8 ຫົວຂໍ້ທີ່ກ່ຽວກັບຫົວຂໍ້ສົນທະນາວິຊາການຢູ່ໃນໂຄງການ) ແລ້ວໃຫ້ເວົ້າກ່ຽວກັບຫົວຂໍ້ນັ້ນໆ, 3) ແມ່ນຈະຕໍ່ຈາກພາກກ່ອນໜ້ານີ້ ແຕ່ຄະນະກຳມະການຈະຖາມເຈາະຈີ້ມກ່ຽວກັບຫົວຂໍ້ນັ້ນ ແລະ 4) ຄະນະກຳມະການ ຈະຖາມກ່ຽວກັບພອນສະຫວັນ ແລະ ຄວາມສາມາດພິເສດ.

-> ການຕອບຄຳຖາມ ແມ່ນບໍ່ຈຳເປັນຕ້ອງຍືດຍາວ ແຕ່ຫາກເປັນຄຳຕອບທີ່ສັ້ນກະທັດຮັດ ແລະ ກົງປະເດັນ. ສຳຄັນທີ່ສຸດ ແມ່ນການສະແດງທັດສະນະຄະຕິເຊີງບວກຕໍ່ຄຳຖາມເຈາະຈິ້ມຈາກຄະນະກຳມະການ ໃນພາກທີ 3 ແລະ ການເວົ້າໃນທຳນອງຂອງຜູ້ທີ່ໄດ້ສຶກສາຄົ້ນຄວ້າ ແລະ ຕິດຕາມຂ່າວສານທັງພາຍໃນ ແລະ ຕ່າງປະເທດທີ່ເປັນສະບັບທາງການຢ່າງເລິກເຊິ່ງ ໃນພາກທີ 2. ສຳລັບພາກທີ 4, ໜຶ່ງໃນທາງທີ່ດີແມ່ນການທຳຄວາມເຂົ້າໃຈກ່ອນວ່າ ໂຄງການຕ້ອງການຄົນທີ່ມີຄຸນສົມບັດ ແລະ ທັກສະໃດແທ້ໆ. ເຮົາມາດສະແດງຈຸດພິເສດ ຫຼື ຄວາມສາມາດຂອງຕົນເອງໄດ້ເຊັ່ນ: ຫຼິ້ນເຄື່ອງດົນຕີ, ຮ້ອງເພງ, ຜົນງານຕ່າງໆທີ່ຕົນໄດ້ສ້າງມາ ແລະ ອື່ນໆ. ການຕອບຄຳຖາມດ້ວຍຄວາມໝັ້ນໃຈ ແລະ ນອບນ້ອມຖ່ອມຕົນໃນເວລາດຽວກັນ ແມ່ນສິ່ງທີ່ຄວນປະຕິບັດສະເໝີ. ການແຕ່ງກາຍໃຫ້ສຸພາບ ກໍແມ່ນສິ່ງທີ່ບໍ່ຄວນມອງຂ້າມເຊັ່ນດຽວກັນ.

-> ຜົນອອກ ແມ່ນປະມານຕົ້ນ ຫຼື ກາງເດືອນມີຖຸນາ ເຊິ່ງຈະປະກອບດ້ວຍເພດຊາຍ 14 ຄົນ ແລະ ເພດຍິງ 14 ຄົນ ລວມທັງໝົດ 28 ຄົນ.

# ຈຳນວນຄົນ ແລະ ເວລາຕ່າງໆນັ້ນ ແມ່ນອາດມີການປ່ຽນແປງ.

# ຊາວໜຸ່ມຜູ້ທີ່ໄດ້ຮັບການຄັດເລືອກ ແມ່ນຈະມາເຝິກຊ້ອມ ແລະ ກຽມຕົວຢູ່ທີ່ສູນກາງຊາວໜຸ່ມປະຊາຊົນປະຕິວັດລາວ (ນະຄອນຫຼວງວຽງຈັນ) ນັບຕັ້ງແຕ່ຊ່ວງຫຼັງວັນທີ່ໄດ້ປະກາດຜົນສຳພາດຜ່ານຈົນເຖິງວັນທີ່ຈະອອກເດີນທາງໄປປະເທດຍີ່ປຸ່ນ.
