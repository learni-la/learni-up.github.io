***ຄະນະຮັບຜິດຊອບກິດຈະກໍາສົນທະນາຫົວຂໍ້ວິຊາການ*** (Discussion Program Steering Committee)

1. ໂຄງຮ່າງການຈັດຕັ້ງ


ຕໍ່ໄປນີ້ ແມ່ນລາຍລະອຽດຂອງ Discussion Program Steering Committee ເຊິ່ງຈັດຕັ້ງຂຶ້ນໂດຍອີງຕາມ “Guide for the 45th Ship for Southeast Asian and Japanese Youth Program in 2018”.


Discussion Program Steering Committee ແມ່ນປະກອບດ້ວຍສະມາຊິກດັ່ງຕໍ່ໄປນີ້:


- ຄະນະນຳພາໂຄງການ (ຫົວໜ້າຄະນະໂຄງການ, ຮອງຫົວໜ້າຄະນະໂຄງການ, ສະມາຊິກຄະນະນຳພາໂຄງການຈຳນວນໜຶ່ງທີ່ຮັບຜິດຊອບກິດຈະກໍາສົນທະນາຫົວຂໍ້ວິຊາການ (Discussion Program))

- ຜູ້ນໍາພາເຮັດກິດຈະກໍາສົນທະນາຫົວຂໍ້ວິຊາການ (Facilitator) 8 ຄົນ (ກຸ່ມລະ 1 ຄົນ)

- ຫົວໜ້າຄະນະຊາວໜຸ່ມ 3 ຄົນ (National Leader (NL)) (ຮັບຜິດຊອບ Discussion Program)

- PY 16 ຄົນ ເປັນຕົວແທນຈາກກຸ່ມສົນທະນາຫົວຂໍ້ວິຊາການ (Discussion Group (DG)) (DG ລະ 2 ຄົນ)

- PY 11 ຄົນ ເປັນຕົວແທນຈາກປະເທດ (ປະເທດລະ 1 ຄົນ)

2. ບົດບາດ


(1) ບົດບາດຂອງ Discussion Program Steering Committee

a. ວາງແຜນ ແລະ ດຳເນີນກິດຈະກໍາສົນທະນາຫົວຂໍ້ວິຊາການ (Discussion Program)

b. ວາງແຜນ ແລະ ຈັດຕັ້ງປະຕິບັດງານແນະນໍາ (Introductory Program) ໃຫ້ແກ່ PY

c. ທຳຄວາມເຂົ້າໃຈ ແລະ ຈັດສັນຄວາມຄືບໜ້າຂອງແຕ່ລະກຸ່ມ

d. ແບ່ງປັນຂໍ້ມູນຂອງແຕ່ລະກຸ່ມ

e. ວາງແຜນ ແລະ ຈັດຕັ້ງປະຕິບັດບົດນຳສະເໜີຂໍ້ສະຫຼຸບຂອງການສົນທະນາ

f. ຮຽບຮຽງບົດລາຍງານການສົນທະນາທັງໝົດເຂົ້າໄວ້ນຳກັນ

g. ວາງແຜນ ແລະ ຈັດຕັ້ງປະຕິບັດວຽກງານເຄື່ອນໄຫວຫຼັງຈົບໂຄງການ (Post-Program Session) (ໂດຍປະສານງານກັນກັບ OBSC)

(2) ບົດບາດຂອງແຕ່ລະຄົນ

a. ຄະນະນຳພາໂຄງການ (Administration)

- ດຳເນີນ Discussion Program Steering Committee (ໃນທ້າຍທີ່ສຸດ, ຄະນະນຳພາໂຄງການ ແມ່ນຈະເປັນຜູ້ຮັບຮອງແຜນການ ແລະ ການດຳເນີນງານຂອງ Discussion Program ແລະ Post-Program Session.)

- ໃຫ້ຄຳແນະນຳກັບ Faci

- ຮັບຮອງເປົ້າໝາຍຂອງແຕ່ລະຫົວຂໍ້ທີ່ສະເໜີໂດຍ Faci

- ຕົກລົງຮັບຮອງຂໍ້ມູນພື້ນຖານ ແລະ ວຽກບ້ານກ່ອນເຂົ້າໂຄງການ (pre-program assignment) ສຳລັບແຕ່ລະກຸ່ມ ເຊິ່ງຈະໄດ້ຮັບການສະເໜີໂດຍ Faci ແລະ ສົ່ງເອກະສານເຫຼົ່ານັ້ນໃຫ້ແກ້ PY ຜ່ານລັດຖະບານຂອງປະເທດຕົນ (ຍົກເວັ້ນຂໍ້ມູນທີ່ເປັນລາຍລະອຽດ)

- ກະກຽມ ແລະ ຈັດແຈງຊັບພະຍາກອນ ແລະ ການຂົນສົ່ງຕ່າງໆທີ່ຈໍາເປັນ

b. ຜູ້ນໍາພາເຮັດກິດຈະກໍາສົນທະນາຫົວຂໍ້ວິຊາການ (Facilitator)

- ກຳນົດ ແລະ ນຳສະເໜີເປົ້າໝາຍຂອງກຸ່ມ ເພື່ອການອະນຸມັດຈາກຄະນະນຳພາໂຄງການ

- ຮ່າງຂໍ້ມູນພື້ນຖານ ແລະ pre-program assignment ເພື່ອການອະນຸມັດຈາກຄະນະນຳພາໂຄງການ

- ປະສານງານ ແລະ ຈັດການກ່ຽວກັບ discussion group (DG) ທີ່ຕົນຖືກມອບໝາຍ

- ນຳພາ DG ແລະ ໃຫ້ການສະໜັບສະໜູນ ແລະ ຄຳແນະນຳໃຫ້ແກ່ PY ໃນກຸ່ມທີ່ຕົນຮັບຜິດຊອບ

- ຮຽບຮຽງ ແລະ ສະຫຼຸບບົດລາຍງານ DG ທີ່ PY ໃນກຸ່ມຕົນເຮັດຂຶ້ນມາ ແລ້ວສົ່ງໃຫ້ກັບຄະນະນຳພາໂຄງການ

- ປະສານງານ ແລະ ຈັດການກ່ຽວກັບວາລະ “Introductory Program,” “ການແນະນຳກ່ຽວກັບການວາງແຜນ ແລະ ການຈັດຕັ້ງປະຕິບັດການເຄື່ອນໄຫວຫຼົງຈົບໂຄງການ (post-program projects design and implementation,” “workshop for post-program projects design and implementation” ແລະ “ການຂຶ້ນນຳສະເໜີຂໍ້ສະຫຼຸບຂອງການສົນທະນາ”

- ເຂົ້າຮ່ວມກອງປະຊຸມກັບ OBSC ເພື່ອສົ່ງຕໍ່ໝາກຜົນຂອງ Discussion Program ລວມທັງໄອເດຍ ແລະ ແຜນສະເໜີຕ່າງໆຂອງ PY ສຳລັບ post-program activity

c. ຫົວໜ້າຄະນະຊາວໜຸ່ມ (National Leader)

- ໃຫ້ຄຳແນະນຳ ແລະ ຄຳຄິດເຫັນທີ່ກ່ຽວກັບແຜນການ ແລະ ການດຳເນີນງານຂອງ Discussion Program ແລະ post-program session ໃນນາມຕົວແທນຂອງຄະນະປະຕິບັດງານເດີນທະເລ (Cruise Operating Committee (COC))

d. ຄະນະຮັບຜິດຊອບທີ່ເປັນ PY (PY Steering Committee member)

- ຊ່ວຍ Faci ດຳເນີນ group discussion

- ປະສານງານກັບ Faci ໃນການກະກຽມ session ແຕ່ລະຄັ້ງ

- ສົ່ງຕໍ່ຂໍ້ມູນຈາກສະມາຊິກກຸ່ມໃຫ້ແກ້ Faci ແລະ ກົງກັນຂ້າມ

- ບັນທຶກຄວາມຄືບໜ້າຂອງການສົນທະນາ ໂດຍປະສານງານກັນກັບ Faci

- ປະສານງານກັບ Faci ໃນການວາງແຜນ ແລະ ດຳເນີນ Introductory Program ແລະ ການຂຶ້ນນຳສະເໜີຂໍ້ສະຫຼຸບຂອງການສົນທະນາ

- ປະສານງານກັບ Faci ໃນການຈັດ “workshop for post-program projects design and implementation”

- ນຳພາ ແລະ ຈັດການກ່ຽວກັບ Post-Program Session III ແລະ IV ໂດຍປະສານງານກັນກັບ OBSC
